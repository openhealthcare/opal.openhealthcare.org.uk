Opal Documentation

Opal is a full stack framework that makes building digital tools for health care easy.

Opal builds deep clinical domain specific functionality on top of Django, Angular and Bootstrap to help developers quickly build easy to maintain, robust clinical applications.

This is the developer documentation.

Opal is an entirely Open Source, Open Governance product, as are the wide library of plugins.

First Steps

If you're new to Opal, then you'll want to take a look at the Installation instructions, and then maybe take the tutorial.
